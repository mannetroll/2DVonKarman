Create a 2D-Turbulence solver in Python = 3.13, in a root package named turbosim.
with a PySide6 GUI showing the changing Vorticity field using Inferno, Gray or Ocean colors,
with controls and statistics grouped in a nice clear way.
Make a fancy black GUI with no flickering numerical data.

With normal decaying Navier-Stokes without any type of forcing or extra terms,
decaying around a moving cylindrical rod with radius R.
Use LS-IMEX-RK3 for time-stepping, and SciPy multithreading for FFT.

Fast rendering: Convert the simulation field to a contiguous uint8 2D NumPy array,
wrap that memory directly in a QImage,
apply a 256-entry color table, then make a QPixmap.

Setup a pyproject.toml project, use hatchling and launch the GUI solver with: 
$ uv run simulation

Controllable Input values:
N = 512, 1024, 2048 (spectral nodes, 3/2 padding)
Re = 10000 (Visc = 1/Re)
CFL = 1.5 (slider from 0.5 to 2.5)
K0 = 5 (int slider from 1 to 25) initial field
NR = 5 (width of rod R, as part of periodic field 2*pi = NR * R)
VR = 0.1 (slider from 0.1 to 10) Velocity of rod in (2*pi/sec) in vertical U-directions.
Field = Vorticity, Energy, U-Velocity, V-Velocity & Stream function
Show frame every n:th step = 2 (or 5 or 10 or 20 or 50)
